modular_backtest.engine.make_initialize(models, factors, commission, slippage, benchmark=None)#

Creates a Zipline initialize function that sets up a pipeline with the given factors, sets the given commission and slippage models, and schedules the given models to be called every day.

Parameters:

  • models (modular_backtest.models.handler.ModelHandler) – The models to be scheduled.

  • factors (modular_backtest.factors.handler.FactorHandler) – The factors to be used in the pipeline.

  • commission (zipline.finance.commission.CommissionModel) – The commission model to be used.

  • slippage (zipline.finance.slippage.SlippageModel) – The slippage model to be used.

  • benchmark (str | None) – The benchmark asset to be used, or None to not use a benchmark.

Returns:

An initialize function that sets up the given models and pipeline.

Return type:

InitializeFunction

class modular_backtest.engine.BacktestEngine(/, **data)#

Bases: pydantic.BaseModel

Runs a backtest of a given set of models.

The models are scheduled to run every day, and the results are stored in a BacktestResult object.

Parameters:

  • models – The models to be scheduled.

  • factors – The factors to be used in the pipeline.

  • capital – The starting capital for the backtest.

  • commission – The commission model to be used.

  • slippage – The slippage model to be used.

  • data (Any)

model_config#

Configuration for the model, should be a dictionary conforming to [ConfigDict][pydantic.config.ConfigDict].

_make_functions()#

Initializes the _initialize and _before_trading_start functions.

Sets up the functions used for initialization and actions before trading starts in the backtest. These functions are generated using the provided models, factors, commission, and slippage settings.
